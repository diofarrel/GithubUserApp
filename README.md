# GithubUserApp
Github User App - Belajar Fundamental Aplikasi Android

**This submission implements :**
1. RecyclerView
2. ViewBinding
3. Parcelable
4. Constraint Layout
5. Explicit and Implisit Intent

## Submission 1

**Features :**
1. Splash Screen
2. List User
3. Detail User
4. Share Item

## Submission 2

3. List Followers and Following
4. Search View

## Submission 3
